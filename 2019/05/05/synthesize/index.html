2、@synthesize作用

  • 生成一个带_的成员变量，如上会自动生成一个_str

  • 为属性生成setter和getter方法（category中除外）；

3、@synthesize使用

  • 配合property属性声明一起使用，如果没有用property声明，没法用synthesize修饰，为什么我们平常只写了property没有写synthesize呢，因为写了property后系统自动添加了synthesize，不需要开发人员手动添加，从而也生成了setter和getter方法；
  • 修改成员变量的名字，默认是加个_，如上例，@synthesize str 默认是@synthesize str = _str，我们可以改成另一个名称，如@synthesize str = str1，代码中用str1访问就好了；
    注意：什么情况下使用synthesize？
  • 想要修改成员变量的名称，用别的名称去访问；
