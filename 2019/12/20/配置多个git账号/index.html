配置多个git账号

Posted on 2019-12-20

mac下配置多个git账户，实现向不同的git仓库提交信息。

1、先查看是否已经设置git的user.name和user.email。

2、设置user.name 和 user.email。如果有设置user.name，user.email可不用再设置，直接第3步。

3、生成ssh key。

4、cd ~/.ssh，进入.ssh文件，在用户/你的用户名/.ssh目录下。

看到此时有id_rsa，id_rsa_pub，id_rsa_xxx，id_rsa_xxx.pub，known_hosts这几个文件，用文本打开id_rsa_xxx文件，复制备用。

5、登录git仓库，配置ssh key。

将上面复制的id_rsa_xxx文件内容粘贴到Key里面，Title自己定义一个就好了。

6、在.ssh文件下新建名为config的文件，文件内容就是配置的git和ssh key。

Host：仓库别名，自己取

Hostname：git repository的真实域名

IdentityFile：对应的id_rsa

注意：配的别名和域名不一样，使用的时候也要把相应的域名改为别名，不同的仓库平台建议别名和域名一致。
